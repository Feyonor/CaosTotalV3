# Caos Total V3 - Mod Fabric 1.21.1

Un mod muy divertido y caótico para jugar con amigos. ¡Explosiones, armas, eventos aleatorios y mucho caos!

## 🎮 Características

### GENERAL
- ✅ Todos los jugadores tienen exactamente 1000 de vida
- ✅ Sin ventajas especiales entre jugadores
- ✅ Comandos especiales solo para administradores
- ✅ Sistema de /home (máximo 5 homes por jugador)
- ✅ Contador de muertes visible

### RECURSOS
- ✅ Muchísimos diamantes en el mundo
- ✅ Pólvora extremadamente fácil de conseguir
- ✅ Redstone extremadamente fácil de conseguir
- ✅ Crafteos muy fáciles

### ARMAS
- ✅ Granada del Caos
- ✅ Lanzacohetes
- ✅ Bazuca
- ✅ Cohetes explosivos

### ARMADURAS
- ✅ Armadura Antiexplosiones (crafteable)
- ✅ Armadura Suprema del Caos (solo de Cajas Sorpresa)

### CAJAS SORPRESA
- ✅ Crafteable fácilmente
- ✅ Generan items aleatorios:
  - TNT
  - Wither
  - Creepers
  - Trampas
  - Diamantes
  - Netherita
  - Objetos especiales
- ✅ Armadura Suprema (extremadamente rara)

### EVENTOS AUTOMÁTICOS
- ✅ Lluvia de TNT
- ✅ Invasiones de mobs
- ✅ Meteoritos
- ✅ Aparición del Rey Creeper
- ✅ Eventos aleatorios

### REY CREEPER
- ✅ Se invoca con TNT + Cabeza de Wither
- ✅ Muchísima vida (500 HP)
- ✅ Invoca mini creepers
- ✅ Hace explosiones enormes

### DIMENSIÓN DEL CAOS
- 🔲 Portal con bloques de diamante (en progreso)
- 🔲 Se activa con Mechero del Caos (en progreso)

## 📋 Requisitos

- **Minecraft**: 1.21.1
- **Fabric Loader**: 0.15.11+
- **Fabric API**: 0.91.0+
- **Java**: 21+

## 🔧 Instalación

1. Descarga el JAR compilado de `Caos Total V3`
2. Colócalo en tu carpeta `mods/` de Minecraft Fabric
3. ¡Listo! Inicia el juego

## 🛠️ Compilación desde Código

```bash
# Clonar repositorio
git clone https://github.com/Feyonor/CaosTotalV3.git
cd CaosTotalV3

# Compilar con Gradle
./gradlew build

# El JAR estará en: build/libs/
```

## 📖 Comandos

### /home
- `/home set <nombre>` - Guardar ubicación actual
- `/home <nombre>` - Teletransportarse a home
- `/home delete <nombre>` - Eliminar home
- `/home list` - Ver todos tus homes

### /caosadmin (Solo administradores)
- `/caosadmin tntrain` - Lluvia de TNT
- `/caosadmin invade` - Invasión de mobs
- `/caosadmin meteorite` - Generar meteoritos
- `/caosadmin summonking` - Invocar Rey Creeper
